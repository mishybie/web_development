LIST METHODS

  • INDEX-returns the index of the first element with the specified value.
  • APPEND-adds an element to the end of the list.
  • COPY-returns a copy of the list.
  • INSERT-adds an element at the specified value.
  • COUNT-returns the number of times an element occurs in a list.
  • POP-removes an element x at an index y.
  • REMOVE-removes the first occurence of an element.
  • SORT-arranges the list in ascending order.

  • example

  • print(thisdict)

PYTHON COLLECTION

  • There are four collection data types in the Python programming language:
  • List-is a collection which is ordered and changeable. Allows duplicate members.
  • Tuple-is a collection which is ordered and unchangeable. Allows duplicate members'
  • Set-is a collection which is unordered, unchangeable*, and unindexed. No duplicate members.
  • Dictionary-is a collection which is ordered** and changeable. No duplicate members.
