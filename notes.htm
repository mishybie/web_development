LIST METHODS

  • INDEX-returns the index of the first element with the specified value.
  • APPEND-adds an element to the end of the list.
  • COPY-returns a copy of the list.
  • INSERT-adds an element at the specified value.
  • COUNT-returns the number of times an element occurs in a list.
  • POP-removes an element x at an index y.
  • REMOVE-removes the first occurence of an element.
  • SORT-arranges the list in ascending order.
